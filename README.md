# Psychotronic Graphics
Realtime 3D graphics renderer.

![Demo](demo.png)

## Features
### Lighting
* [x] Physically-based rendering (PBR)
* [x] Image-based lighting (IBL)
* [ ] Reflection probes
* [ ] Shadow mapping
* [x] Skybox
### Acceleration
* [ ] Frustum culling
* [ ] Occlusion culling
* [ ] Clustered forward rendering
